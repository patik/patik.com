Chromorph

Color Converter

Chromorph is a color conversion tool aimed at web developers dealing with colors in CSS. It can be used as a standalone function or (coming soon) a jQuery plugin.

Features

Standard library and jQuery Plugin

  • Converts alpha-transparent colors to their equivalent opaque formats (hex, rgb) while taking the background into account
  • Provides a range of output options:
    • "rgb(12,34,56)"
    • [12, 34, 56]
    • {rgb: { string: 'rgb(12,34,56)', array: [12,34,56]}, hex: ...}
